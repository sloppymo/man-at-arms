# Artwork Directory

Place your medieval manuscript artwork images here.

## Image Files Needed:

1. **character-creation.jpg** - Medieval town square scene (for character creation)
2. **battle-scene-1.jpg** - French vs English battle scene (for first_battle_brave)
3. **battle-scene-2.jpg** - Mountainous battle scene (for battle_crecy)
4. **naval-battle-1.jpg** - Naval battle scene (for channel_crossing)
5. **naval-battle-2.jpg** - Naval battle with coastal city (for saint_vaast_landing)
6. **naval-battle-3.jpg** - Another naval battle scene (for landing_raid)

## Usage:

Images are referenced in scene definitions with the `artwork` property.
Example: `artwork: "artwork/battle-scene-1.jpg"`

All artwork will display above the story text with a fade-in animation.
