# Character Creation Step-Based System

This document outlines the step-based character creation system.

## Steps:
1. **Step 1: Name & Region** - Name input and England map selection
2. **Step 2: Age Range** - Age range selection with advantages/drawbacks
3. **Step 3: Origin Story** - Background origin selection
4. **Step 4: Background Questions** - Optional background questions
5. **Step 5: Priorities** - Priority system (A-E assignment)
6. **Step 6: Review & Begin** - Final review and start game

## Navigation:
- Previous/Next buttons at bottom of each step
- Step indicator at top showing current step (1/6, 2/6, etc.)
- Can navigate back to previous steps to make changes

## England Map:
- Vector SVG map of England with accurate coastline
- 8 clickable regions: Yorkshire, Kent, London, Cornwall, Lancashire, Essex, Norfolk, Somerset
- Hover tooltips with region flavor text
- Visual feedback on selection
